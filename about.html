About CarFinder

CarFinder uses the Edmunds API to get up to date car information. This includes makes, models, detailed information on cars, and dealer information. Also CarFinder uses the google maps api to display a detailed interactive map of the dealers location. On top of that the website is built for mobile devices using Jquery Mobile.

The creator of the CarFinder web application is Matt Kleinberg, a senior Digital Media student at the University of Central Florida.
